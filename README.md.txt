## üé® Customization
 
### Colors
The color scheme is defined in CSS custom properties in `styles.css`:
 
```css
:root {
    --primary-green: #2d5016;
    --primary-green-light: #4a7c2c;
    --accent-yellow: #f4a825;
    --accent-yellow-light: #f7c04e;
    /* ... more colors */
}
```
 
### Typography
Fonts can be changed by updating the Google Fonts link in `index.html` and the CSS variables:
 
```css
:root {
    --font-heading: 'Playfair Display', serif;
    --font-body: 'Inter', sans-serif;
}
```
 
### Content
- Update text directly in `index.html`
- Modify product information in the Products section
- Change contact details in the Contact section
